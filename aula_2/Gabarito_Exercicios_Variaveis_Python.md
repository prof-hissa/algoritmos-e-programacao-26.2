# Gabarito - Exercícios de Variáveis em Python

## 1. Variável inteira (idade)

``` python
idade = 25
print("Idade:", idade)
```

------------------------------------------------------------------------

## 2. Variável decimal (altura) formatada

``` python
altura = 1.75
print(f"Altura: {altura:.2f} metros")
```

------------------------------------------------------------------------

## 3. Constante PI

``` python
PI = 3.14159
print("Valor de PI:", PI)
```

------------------------------------------------------------------------

## 4. Soma de dois números inteiros

``` python
num1 = 10
num2 = 20
soma = num1 + num2
print("Soma:", soma)
```

------------------------------------------------------------------------

## 5. Multiplicação de dois números decimais

``` python
num1 = 5.5
num2 = 2.0
multiplicacao = num1 * num2
print("Multiplicação:", multiplicacao)
```

------------------------------------------------------------------------

## 6. String com mensagem de boas-vindas

``` python
nome = "Wilson"
print("Bem-vindo,", nome + "!")
```

------------------------------------------------------------------------

## 7. Variável caractere (sexo)

``` python
sexo = "M"
print("Sexo:", sexo)
```

------------------------------------------------------------------------

## 8. Valor total da compra

``` python
valor_produto = 50.0
quantidade = 3
total = valor_produto * quantidade
print("Total da compra: R$", total)
```

------------------------------------------------------------------------

## 9. Aplicando desconto com constante

``` python
TAXA_DESCONTO = 0.10  # 10%
valor = 200.0
valor_final = valor - (valor * TAXA_DESCONTO)
print("Valor final com desconto: R$", valor_final)
```

------------------------------------------------------------------------

## 10. Média aritmética de três notas

``` python
nota1 = 7.5
nota2 = 8.0
nota3 = 9.0
media = (nota1 + nota2 + nota3) / 3
print("Média:", media)
```

------------------------------------------------------------------------

## 11. Salário líquido

``` python
salario_bruto = 3000.0
desconto = 500.0
salario_liquido = salario_bruto - desconto
print("Salário líquido: R$", salario_liquido)
```

------------------------------------------------------------------------

## 12. Área do retângulo

``` python
base = 10
altura = 5
area = base * altura
print("Área do retângulo:", area)
```

------------------------------------------------------------------------

## 13. Conversão Celsius → Fahrenheit

``` python
celsius = 30
fahrenheit = (celsius * 9/5) + 32
print("Temperatura em Fahrenheit:", fahrenheit)
```

------------------------------------------------------------------------

## 14. Cálculo de salário por hora

``` python
horas_trabalhadas = 160
valor_hora = 25.0
salario = horas_trabalhadas * valor_hora
print("Salário calculado: R$", salario)
```

------------------------------------------------------------------------

## 15. Velocidade média

``` python
distancia = 300  # km
tempo = 5        # horas
velocidade_media = distancia / tempo
print("Velocidade média:", velocidade_media, "km/h")
```
